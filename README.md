# xsap

#### 介绍
X+ Student appointment platform, X+学员约课平台，主要是针对K12的培训机构的会员、课程、约课安排的管理平台，支持操作人员后台约课、排课、日程计划等工作，也支持微信公众号由客户在线预约处理。
